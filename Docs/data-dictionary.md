# Data Dictionary — Lending Club Credit Risk Project

## Source
Lending Club Loan Data (2007–2018)
Downloaded from: https://www.kaggle.com/datasets/wordsforthewise/lending-club

## Files
| File | Rows | Size | Description |
|------|------|------|-------------|
| accepted_2007_to_2018Q4.csv | 2,260,701 | 1.55GB | All funded loans |
| rejected_2007_to_2018Q4.csv | 27,648,741 | 1.65GB | All rejected applications |

## Key Business Metrics
- **Default Rate:** % of loans with status "Charged Off"
- **DTI:** Debt-to-Income ratio — monthly debt ÷ monthly income
- **FICO Score:** Credit score ranging 300–850, higher = lower risk
- **Loan Grade:** A (lowest risk) to G (highest risk)
